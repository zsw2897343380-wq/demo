Here's a complete, well-documented Java implementation of a Hello World program following best practices:

```java
/**
 * A comprehensive Hello World program demonstrating Java best practices.
 * 
 * This class provides multiple ways to display "Hello, World!" including:
 * - Basic console output
 * - Formatted output
 * - Method abstraction
 * - Unit test friendly design
 */
public class HelloWorld {

    // Private constructor to prevent instantiation of utility class
    private HelloWorld() {
        throw new UnsupportedOperationException("Utility class cannot be instantiated");
    }

    /**
     * Returns the standard "Hello, World!" greeting string.
     * 
     * @return The greeting string "Hello, World!"
     */
    public static String getGreeting() {
        return "Hello, World!";
    }

    /**
     * Returns a personalized greeting for the given name.
     * 
     * @param name The name to include in the greeting
     * @return A personalized greeting string
     * @throws IllegalArgumentException if name is null or empty
     */
    public static String getGreeting(String name) {
        if (name == null || name.trim().isEmpty()) {
            throw new IllegalArgumentException("Name cannot be null or empty");
        }
        return String.format("Hello, %s!", name.trim());
    }

    /**
     * Prints the standard greeting to standard output.
     */
    public static void printGreeting() {
        System.out.println(getGreeting());
    }

    /**
     * Prints a personalized greeting to standard output.
     * 
     * @param name The name to include in the greeting
     */
    public static void printGreeting(String name) {
        System.out.println(getGreeting(name));
    }

    /**
     * Main method - entry point for the application.
     * Demonstrates various ways to use the HelloWorld class.
     * 
     * @param args Command line arguments (optional name for personalized greeting)
     */
    public static void main(String[] args) {
        System.out.println("=== Hello World Demo ===");
        System.out.println();

        // Basic greeting
        System.out.println("1. Basic greeting:");
        printGreeting();

        System.out.println();

        // Personalized greeting with command line argument
        System.out.println("2. Personalized greeting:");
        if (args.length > 0) {
            printGreeting(args[0]);
        } else {
            printGreeting("World");
        }

        System.out.println();

        // Demonstrate the getGreeting method
        System.out.println("3. Using getGreeting() method:");
        String greeting = getGreeting();
        System.out.println("Greeting string: " + greeting);

        System.out.println();

        // Demonstrate formatted output
        System.out.println("4. Formatted output:");
        System.out.printf("The greeting '%s' has %d characters.%n", 
                         greeting, greeting.length());

        System.out.println();

        // Demonstrate exception handling
        System.out.println("5. Error handling demonstration:");
        try {
            getGreeting(null);
        } catch (IllegalArgumentException e) {
            System.out.println("Caught expected exception: " + e.getMessage());
        }

        System.out.println();
        System.out.println("=== End of Demo ===");
    }
}
```

## Usage Examples

### 1. Basic Usage (Command Line)
```bash
# Run with default greeting
java HelloWorld

# Run with personalized greeting
java HelloWorld Alice
```

### 2. Using in Another Class
```java
public class Application {
    public static void main(String[] args) {
        // Get greeting string
        String greeting = HelloWorld.getGreeting();
        System.out.println(greeting);
        
        // Get personalized greeting
        String personalGreeting = HelloWorld.getGreeting("Bob");
        System.out.println(personalGreeting);
        
        // Print directly
        HelloWorld.printGreeting();
        HelloWorld.printGreeting("Charlie");
    }
}
```

### 3. Unit Test Example
```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

public class HelloWorldTest {
    
    @Test
    void testGetGreeting() {
        assertEquals("Hello, World!", HelloWorld.getGreeting());
    }
    
    @Test
    void testGetGreetingWithName() {
        assertEquals("Hello, Alice!", HelloWorld.getGreeting("Alice"));
    }
    
    @Test
    void testGetGreetingWithNull() {
        assertThrows(IllegalArgumentException.class, 
                    () -> HelloWorld.getGreeting(null));
    }
    
    @Test
    void testGetGreetingWithEmpty() {
        assertThrows(IllegalArgumentException.class, 
                    () -> HelloWorld.getGreeting(""));
    }
}
```
